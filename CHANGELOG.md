## [1.0.0] - 2024-11-26

### Added
- Initial release
- `DurationUtil` service for parsing and formatting durations
- Support for multiple duration input formats (HH:MM, HH:MM:SS, human-readable)
- `DurationType` form type for Symfony forms
- Twig extension with `duration`, `toSeconds`, and `hhmm` filters
- DateInterval conversion utilities
- Comprehensive unit tests
- Full documentation and examples
- GitHub Actions CI workflow
